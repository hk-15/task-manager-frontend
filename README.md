# Task Manager Frontend
## Introduction
This is a React TypeScript frontend for interacting with the Task Manager backend API, which can be found in this repository: [https://github.com/hk-15/task-manager-backend](https://github.com/hk-15/task-manager-backend). Users can add tasks to the database using the form and the UI will display the newly created task.

## Installation Guide
* Clone this repository [https://github.com/hk-15/task-manager-frontend](https://github.com/hk-15/task-manager-frontend).
* Run `npm install` to install all dependencies.

## Usage
* Run `npm run dev` to run the app in development mode. You can view the app at [http://localhost:5173/](http://localhost:5173/). If you make edits, the page will reload.
* Run `npm run build` to build the application ready for deployment.
* Run `npm test` to run the tests.
* Run `npm run lint` to run the linter.
